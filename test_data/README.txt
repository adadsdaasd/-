==================================================
            测试数据文件说明 (v2.2)
==================================================

本文件夹包含用于测试 Digital Twin 应用的虚拟数据。
** 所有数据都包含系统必填字段，可直接使用 **

【系统必填字段】
- 姓名
- 电话（用于身份识别和去重）
- 学历（教育背景）
- 工作经历 或 工作年限
- 技能特长

==================================================
                    测试流程
==================================================

【测试1: 个人版数据导入】
1. 切换到"个人版"模式
2. 在侧边栏选择"导入数据"
3. 上传"个人档案_李明.csv"
4. 或粘贴"自我介绍_李明.txt"内容进行AI分析
5. 验证数据显示完整（含技能特长）

【测试2: 多人版数据导入】
1. 切换到"多人版"模式
2. 在侧边栏选择"导入数据"
3. 上传"team_5_members.xlsx"
4. 验证5人数据都成功导入，每人都有技能特长

【测试3: 智能填表 - 一人一表】
1. 进入"智能填表"功能
2. 上传"blank_job_application.xlsx"
3. 选择已导入的档案
4. 点击"AI识别字段"
5. 点击"批量生成答案"
6. 检查技能特长等字段是否正确填写

【测试4: 智能填表 - 一表多人】
1. 切换到"多人版"模式
2. 进入"智能填表"功能
3. 上传"blank_team_summary.xlsx"
4. 选择"按团队选择"或手动多选
5. AI自动检测为"一表多人"模式
6. 批量生成并导出汇总表

【测试5: 问答式填表】
1. 进入"智能填表"功能
2. 选择"粘贴问题列表"
3. 粘贴"问题列表_面试问答.txt"内容
4. 选择档案并生成答案
